# Внешние плагины

Складывайте сюда пользовательские плагины (`*.py`).

Каждый плагин должен экспортировать функцию `get_plugin()` и возвращать объект с полями:

- `plugin_id`
- `title`
- async-метод `transform(text: str, context) -> str`

## Пример

В репозитории уже есть рабочий пример: `plugins/example_reverse.py`.

После запуска `telethon-fancifier setup` модуль появится в списке доступных и его можно назначить на любой выбранный чат.
